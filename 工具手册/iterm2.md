### 一些快捷键
- 分屏相关的
```bash
command + d             # 垂直分屏
command + shift + d     # 水平分屏
command + [ 或 ]        # 切屏
```
- 标签相关
```bash
command + t                 # 新建标签
command + w                 # 关闭标签
command + 方向键 或 数字键    # 切换标签
command + enter            # 切换全屏
```
- 其他操作
```bash
command + f             # 搜索
ctrl + u                # 清除当前行
ctrl + a                # 到行首
ctrl + e                # 到行尾
ctrl + r                # 搜索命令
ctrl + k                # 删除光标后面的字符
ctrl + h                # 删除光标前面的字符
ctrl + d                # 删除光标当前的字符
```
